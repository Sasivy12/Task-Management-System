# Установка и запуск
1. Клонирование репозитория:

```git clone https://github.com/Sasivy12/Task-Management-System.git```

2. Настройка переменных окружения:
```
POSTGRES_DB=task_management
POSTGRES_USER=admin
POSTGRES_PASSWORD=secret
```
3. Запуск контейнеров:
   
```docker-compose up```

4. Перейти по адресу:
   
```http://localhost:8080```
